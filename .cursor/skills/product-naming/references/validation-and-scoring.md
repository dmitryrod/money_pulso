# Validation and brandability scoring

## Шкала (1–10) по осям

Оцени каждого финалиста по пяти осям (кратко, 1 строка обоснования на ось):

| Axis | Что оцениваем |
|------|----------------|
| Memorability | Короче и отличимее — обычно выше; штраф за клише ниши |
| Pronounceability | Удобно произнести вслух на целевых языках |
| Spellability | Предсказуемое написание после устного упоминания |
| Uniqueness | Не смешивается с конкурентами и generic terms |
| Domain fit | Насколько имя *логично* сочетается с желаемым TLD (не «доступность») |

**Не прибавляй баллы** за «домен точно свободен» без реальной проверки.

## Быстрый чеклист перед рекомендацией

- Легко написать по слуху и прочитать вслух без тренировки.
- Нет явных негативных ассоциаций на целевых языках/рынках.
- Не очевидный конфликт с известным брендом в той же категории (эвристика; не юридическое заключение).
- Нет запрещённых пользователем морфем/паттернов.
- Для B2B: имя не слишком «игривое», если контекст требует серьёзности (или наоборот — согласовано с voice).

## Итоговый shortlist

После скоринга:

1. Топ **3–5** с суммарным комментарием «почему не остальные».
2. Один **primary recommendation** и один **runner-up**.
3. Явный блок **риски и проверки** (домен, TM, соцсети).
