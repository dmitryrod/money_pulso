# Molecules

## Summary

- Molecules are reusable section patterns.
- Use canonical titles and canonical content shapes.

## Canonical molecules

- **Summary Block** ‚Äî `## Summary`; 3-6 bullets.
- **Overview Block** ‚Äî `## Overview`; purpose, audience, scope bullets.
- **Rules Block** ‚Äî `## Rules`; one rule per bullet.
- **Workflow Block** ‚Äî `## Workflow`; ordered steps.
- **Variables Block** ‚Äî `## Variables`; `- **name** ‚Äî value` only.
- **Interfaces Block** ‚Äî `## Interfaces`; per interface: Purpose, Inputs, Outputs, Constraints.
- **Metadata Block** ‚Äî `## Metadata`; key-value bullets.
- **Example Block** ‚Äî `## Example`; one intro line + one fenced snippet.
- **Decision Block** ‚Äî `## Decision`; context, options, chosen outcome.
- **Notes Block** ‚Äî `## Notes`; max 5 bullets.
- **Warnings Block** ‚Äî `## Warnings`; downside/failure bullets.
- **Tips Block** ‚Äî `## Tips`; concise practical bullets.
- **Reference Block** ‚Äî `## Reference index`; link bullets.
- **Checklist Block** ‚Äî `## Checklist`; task list syntax.
- **FAQ Block** ‚Äî `## FAQ`; `### Question` + short answer.
- **Comparison Block** ‚Äî `## Comparison`; `### Option A/B` with bullets.
- **Footer Block** ‚Äî terminal region; max 2 lines.

## Rules

- Do not invent ad-hoc molecule names in files.
- If a needed molecule is missing, add it here first.
